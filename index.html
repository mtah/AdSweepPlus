AdSweep+ is a continuation of the AdSweep ad-removal utility for Chromium-based browsers (e.g. Google Chrome) as well as Opera, Safari and Firefox* and other Greasmonkey-capable browsers.

Get it

  • Chromium-based browsers: Download and install the Chromium extension.

  • Other browsers: Theoretically, any browser capable of running Greasemonkey user scripts is capable of running AdSweep+:

    • Opera: built-in support
    • Safari: GreaseKit
    • Firefox: GreaseMonkey extension

    Download AdSweepPlus.js.

Note: This section is incomplete. For now, refer to the documentation of each user scripting client and the original AdSweep site for detailed installation instructions.

Contribute

I'm picking this project up with the intention of having other people contribute to it as well (ad rules, bug fixes, you name it!). If you want to (and please do), the preferred way is to fork the project on GitHub, make your changes and send me a pull request.

*I'm on Firefox, should I use this?

Firefox already has a very popular native ad-removal extension called AdBlock+. Thus, while that extension is in active development, no active effort will be put into testing AdSweep+ with Firefox. You are recommended to use AdBlock+ instead.
